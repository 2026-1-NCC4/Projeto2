# Análise Inferencial de Dados

## 1. Visão Geral
<p>Esta entrega apresenta análises inferenciais aplicadas aos dados da Cannoli, utilizando regressão linear, testes estatísticos e visualizações para identificar relações entre variáveis e apoiar tomadas de decisão.</p>

## 2. Metodologia
* **Ambiente:** RStudio  
* **Pacotes:** `readr`, `dplyr`, `lubridate`.

## 3. Arquivos Utilizados
<p>Foram utilizados os arquivos <b>STOREORDER</b>, <b>CAMPAIGN</b> e <b>CAMPAIGNxORDER</b>, além do dicionário de dados disponibilizado para interpretação das variáveis.</p>
